alert창 사용방법

JavaScript의 alert() 함수는 웹 페이지에 팝업 창을 표시하여 메시지를 보여줍니다. 다음은 기본적인 alert() 함수 사용 예입니다.

						alert("안녕하세요! 이것은 알림창입니다.");

위 코드를 실행하면 웹 페이지 중앙에 "안녕하세요! 이것은 알림창입니다."라는 메시지가 담긴 팝업 창이 나타납니다. 사용자는 "확인" 버튼을 클릭하여 팝업 창을 닫습니다.

주의 사항:

  • alert() 함수는 비동기적으로 실행됩니다. 즉, 알림창이 표시되고 사용자가 확인 버튼을 클릭할 때까지 다른 코드가 실행되지 않습니다.
  • alert() 함수는 단순한 메시지를 표시하는 데만 사용될 수 있습니다. 사용자 입력을 받거나 복잡한 작업을 수행하기 위해서는 다른 방법 (예: prompt(), confirm()) 을 사용해야 합니다.

추가 예제:

					// 변수에 메시지를 저장하고 alert 함수에 전달
	let message = "이것은 변수에 저장된 메시지입니다.";
	alert(message);

	// alert 함수를 다른 함수 내에서 사용
	function showMessage() {
  alert("안녕하세요! 이것은 함수 내부에서 실행되는 알림창입니다.");
	}
	showMessage();
CSS 배경색 변경하기

CSS로 배경색 변경하기

CSS를 사용하여 웹 페이지의 배경색을 변경하는 방법은 여러 가지가 있습니다. 어떤 방법을 사용할지는 웹 페이지의 구조와 디자인 요구 사항에 따라 결정됩니다.

1. 전체 페이지 배경색 변경:

					body {
					  background-color: #yourColorCode; /* 예: #FF0000 (빨강), #00FF00 (초록), #0000FF (파랑) */
					}

2. 특정 요소의 배경색 변경:

/* 예: h1 태그의 배경색을 노란색으로 설정 */
					h1 {
					  background-color: yellow;
					}

3. 배경 이미지 사용:

body {
					  background-image: url("yourImagePath.jpg"); /* 이미지 경로를 입력 */
					  background-repeat: no-repeat; /* 이미지 반복 방지 */
					  background-size: cover; /* 이미지 화면 전체에 맞춤 */
					}

4. 그라디언트 배경 사용:

body {
					  background: linear-gradient(to right, #FF0000, #FFFF00); /* 빨강에서 노랑으로 변화하는 그라디언트 */
					}

참고 사항:

  • #yourColorCode는 원하는 색상 코드를 입력해야 합니다. 16진수 코드(예: #FF0000) 또는 색상 이름(예: red, green, blue)을 사용할 수 있습니다.
  • CSS 속성은 웹 페이지의 HTML 요소에 적용됩니다. 따라서 어떤 요소의 배경색을 변경하고 싶은지 명확하게 지정해야 합니다.
  • 다양한 CSS 속성을 조합하여 원하는 효과를 얻을 수 있습니다.
